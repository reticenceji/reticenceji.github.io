s[i:j]s 从 i 到 j 的切片
s[i:j:k]s 从 i 到 j 步长为 k 的切片
len(s)s 的长度

range类型是不可变的数字序列
range(start,end,step)	#range类型的构造器函数

我们知道python中的列表变量，其实是对列表的一个引用(reference).在下面的例子中，对b的修改同样会对a造成改变。我们可以用copy模块的deepcopy和copy函数避免这一现象。

import copy
a = [1,2,[3,4]]
b = a
c = copy.copy(a)        #把引用里的内容都复制一遍，形成一个新的列表
d = copy.deepcopy(a)    #deepcopy()函数将同时复制它们内部的列表。递归的把引用里的引用的内容都复制了

Dict

Python内置了字典：dict的支持，dict全称dictionary，在其他语言中也称为map，使用键-值（key-value）存储，具有极快的查找速度。但需要占用大量的内存。

d = {'URL': 'www.baidu.com', 'Code': 202}	
d['Body'] = '123'	#添加键值对
d.get("URL")	#返回key对应的value，并且在key不存在时返回None
d.values(),d.keys()		#顾名思义 
d.items()	#返回的是所有的键值对tuple
#上面两个方法返回的都是视图对象view objects.该对象提供字典条目的一个动态视图。

dict的key必须是不可变对象，因为dict通过计算key的Hash来确定value的地址。

\s空白字符
\S非空白字符
\w单词字符
\W非单词字符
\.匹配.号,\是转义字符
数量限定字符内容
*匹配前一个字符0次或无限次
+匹配前一个字符1次或无限次
?匹配前一个字符0次或1次
{m}匹配前一个字符m次
{m,n}匹配前一个字符m-n次
位置限定字符内容
(exp)匹配exp并捕获到自动命名的组中
(? exp)匹配exp并捕获到名为name的组中
(?:exp)匹配exp但是不捕获匹配的文本
(?=exp)匹配exp前面的位置
(?<=exp)匹配exp后面的位置
(?!exp)匹配后面不是exp的位置
(?<!exp)匹配前面不是exp的位置

re.search(pattern, string, flags=0)		# 扫描整个 字符串 找到匹配样式的第一个位置，并返回一个相应的 匹配对象。如果没有匹配，就返回一个 None ； 注意这和找到一个零长度匹配是不同的。
#和match的区别在于Search可以从任一一个位置开始匹配.

re.finditer(pattern, string, flags=0)
#pattern 在 string 里所有的非重复匹配，返回为一个迭代器 iterator 保存了 匹配对象 。 string 从左到右扫描，匹配按顺序排列。空匹配也包含在结果里。

import re
text = """<p><strong>说明：</strong> 如果需要匹配的字符是正则表达式中的特殊字符，那么可以使用\进行转义处理，例如想匹配小数点可以写成\.就可以了，因为直接写.会匹配任意字符；同理，想匹配圆括号必须写成\(和\)，否则圆括号被视为正则表达式中的分组。</p>
</blockquote>
<h3 id="h3-python-"><a name="Python对正则表达式的支持" class="reference-link"></a><span class="header-link octicon octicon-link"></span>Python对正则表达式的支持</h3><p>Python提供了re模块来支持正则表达式相关操作，下面是re模块中的核心函数。</p>"""

reg = re.compile("<p>(.*)</p>")
