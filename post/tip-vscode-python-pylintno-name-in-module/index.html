Tip-VScode Python pylint(no-name-in-module)

2021-04-25 1 min read # Python # Tools

安装第三方包导入模块的时候，出现了这样的错误提示：
No name 'QApplication' in module 'PyQt5.QtWidgets'pylint(no-name-in-module)
一开始以为是VSCode没有找到第三方包，但实际上是pylint的问题。
新版pylint默认不支持外部扩展模块,pyqt5是C++写的外部扩展，因此在【用户设置】那里添加下面设置即可：

"python.linting.pylintArgs": [
        "--extension-pkg-whitelist=PyQt5"
    ]

别的第三方包也是同理。
