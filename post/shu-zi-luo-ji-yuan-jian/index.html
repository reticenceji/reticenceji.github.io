2. 逻辑函数的化简

我们有三个判断化简程度的标准：

  • 文字成本（Literal cost，简称L）：对表达式中所有的文字计数即可。
  • 门输入成本（gate-input cost，简称G）：与等式一一对应的实现中所用门输入端的个数。那么怎么从等式中计算呢？首先计算所有的文字，然后再加上所有运算(不包括非)所代表的门个数，再-1。
  • 包括非门的门输入成本，称为GN。

我们可以用公式法化简。公式法没有办法判断是否化到了最简，只能尽力而为。

我不知道怎么表达卡诺图的化简办法，只能说卡诺图中所有相邻的项都有公共因子，可以组合起来。我们首先确定所有的prime，然后对所有的essential作和，在用prime包含剩下的，就完成了化简。这个还是看书和动手吧，需要熟练，我的水平讲不清楚。
卡诺图化简有几个概念：

  • 方格(rectangle)：在卡诺图中指的是包含2^m个小方格的矩形。
  • 蕴含项(implicant)：函数对某个乘积项的所有最小项取值为1，成为蕴含项。在卡诺图中就是由1组成的方格。
  • 主蕴含项(prime implicant)：如果从该乘积项P中移除任何一个变量，得到的乘积项不再是函数的蕴含项，则P称为主蕴含项。在卡诺图中就是由1组成的尽可能大的方格。
  • 必要主蕴含项/质主蕴含项(essential prime implicant)：如果存在一个1的最小项，仅存在于某个主蕴含项中，那么这个主蕴含项就是必要主蕴含项。
  • 无关最小项(don't care condition)：在实际情况中，某些输入组合的取值我们不关心他是0还是1,我们用X来表示。他可以辅助化简，即可以当成0又可以当成1。我们用d表示。

3. 组合逻辑电路的设计

设计一个数字系统的步骤为。

  1. Specification。指定所需的行为。有几种输入、输出，并为他分配变量。
  2. Formulation。以布尔方程或真值表的方式对系统的输入和输出进行形式化。
  3. Optimization。优化（化简），比如两极电路的化简可以利用卡诺图。
  4. Technology Mapping。工艺映射。工艺映射指的是将逻辑图转换成可以用工艺实现的新的图。例如我们只有与非门，那我们要把我们用到的元器件都用与非门进行替换，但仍然保持原来的功能。我们得到设计的电路图。
  5. Verification。验证设计的正确性。这个非常复杂。

逻辑电路的设计主要分成 组合逻辑电路设计和时序电路设计，组合逻辑电路的状态只和输入有关，和上一个状态无关。

时序逻辑电路分析：根据电路从第一层向后列出每个门的方程，化简。这样就可以得到输出的方程。

1.1 锁存器 Latch

锁存器是满足下面三个条件的元器件

  1. 可以维持一个稳定状态。
  2. 稳定状态是0或1。
  3. 在特定条件下，状态可以再0和1之间切换。

锁存器的一般用途是用来组成触发器。
锁存起存在着空翻问题，就是在一个周期内，输出的值可以发生多次变化。以D Latch为例，当C=1的时候，输出Q的值可以随着D的变化发生多次变化。

SR Latch

或非门组成的SR Latch，是1有效。用与非门也可以组成Latch。
注意到当S=R=1的时候，两个输出都是0，这是非法的输出。

D Latch

为了防止S=R=1这种非法输出，引入D Latch。C可以是控制信号Control，也可以是时钟信号Clock。

  • C=1的时候，右边SR锁存器的输入取决于D。

2.1 触发器 FlipFlop

通常情况下，Flipflop是由Latch组成的。构造触发器通常用两个Latch构成Master-Slave Flip-flop。
触发器有两个输出端，一个存储正常值，一个存储反值。
我们有几个时间参数Timing来评价一个触发器：

  • setup time：tst_sts​在时钟触发沿之前，input必须保持不变的最小时间。小于可能对输出造成错误。
  • hold time：tht_hth​在时钟触发沿之后，input必须保持不变的最小时间。小于可能对输出造成错误。
  • minimum clock pulse width：min(tw)min(t_w)min(tw​)最小的时钟脉冲宽度。小于锁存器没有足够的时间捕获输入值。
  • propagation delay time：tpdt_{pd}tpd​时钟触发沿和输出为一个稳定的新值之间的时间间隔。

SR Master-Slave Flipflop

使用较少。存在一次性采样问题（1s catching）。就是说在C=1的时候，我们的输入是一直可以改变并改变Y的值的。但是最后被传递给Q的只有在t=1->0时刻的值。如红框中的部分，S偶然有一个尖峰，此时C=1，那么Y就被设置成1并保存下来。然后在t=1->0的时候传递给了输出，就得到了错误的结果。这个也是边沿触发。再入红框左边的部分。还存在一个问题，就是R=S=1的非法输入问题。

Negative-Edge-Triggered D Flip-Flop

D锁存器是目前使用最为广泛的flip-flop。
考虑输出随时钟变化的情况：

  • 当时钟为0时，Master不变，Slave和Master保持一致，由于Master不变所以Slave也不变。
  • 从0到1时，Master开启，Slave关闭。
  • 当时钟为1时，Master和输入保持同步，但是此时Slave关闭，保持不变。
  • 从1到0，此时Master关闭，Slave打开，Master的值被复制到Slave中。

所以，我们的输出只有在负边沿才会改变，在除此之外的时钟周期保持稳定。如果我们想改变输出，应该在高电平时给他一个稳定的输入。

  • J=1, K=0：Set
  • J=0, K=1：Reset
  • J=1, K=1：取反
  • J=0, K=0：保持

时序电路的行为是由电路的输入input，输出output和状态state决定的。
我们对时序电路的分析，就是要确定输入输出和状态的时间序列。

同步时序电路的结构和功能，可用三组逻辑函数表达式描述。

  1. 输出output函数表达式：是一组反映电路输出Z与输入x和现态y之间关系的表达式。
    Zi = fi(x1，…，xn ,y1,…，ys) i=1,2,…,m（Mealy型电路，输出和输入有关)
    Zi = fi(y1，…，ys)　　　　 i=1,2,…,m（Moore型电路，输出和输入无关）

  2. 激励excitation函数表达式： 激励函数又称为控制函数，它反映了存储电路的输入Y与外部输入x和电路状态y之间的关系。其函数表达式为
    存储电路的输入Y = g(外部输入X,电路状态y)

  3. 次态函数(Next state function)表达式：
    次态y(n+1) = f(存储电路输入Y,现态y(n))

第五章 寄存器和寄存器传输

寄存器register：一个寄存器包含一组触发器。由于每一位触发器可以保存1 bit的信息，因此由n个触发器组成的n bit寄存器可以保存n bit的信息。除了一组触发器外，寄存器还要包含实现其状态转换的门电路。
计数器counter：计数器是指在时钟脉冲的激励下，能遍历预先规定好的状态序列的一种寄存器。常见的计数序列有模N计数器modulo N、BCD计数器。

加载

加载loading：把新信息传送至寄存器称为寄存器的加载。如果所有的位都是在公共时钟脉冲同时加载的，我们称之为并行加载 parallel。
并行加载有几种实现方式。
第一种非常简单，就是给时钟信号与上一个Load位。这样当Load=0的时候，时钟信号被禁止，我们不能加载。这种技术被称为门控时钟clock gating。
但是这种技术是存在弊端的，在时钟脉冲的路径上插入额外的门，将会带来延迟，或者说是时钟偏移clock skew。所以这种方法不建议使用，尤其是在高速系统上。
第二种方法其实是利用一个1-2选择器。选择加载的是输出Q还是新的输入D。虽然电路复杂了一点，但是比较好。

2. 计数器Counter

计数器分为两个类型：
行波计数器Ripple Counter：某些触发器的输出值的改变可以改变其他触发器中的值，也就是说某些触发器C输入端的值不是公用的时钟脉冲，而是其他触发器的输出信号。可以看到他十分简单，但是同样的时钟不统一会增加电路的不确定性。

同步计数器Synchronous Binary Counters：所有的C输入端都是公用的时钟脉冲，并且计数器下一状态的值取决于当前所处的状态。
串行serial同步计数器，和行波计数器逻辑相似。不同的是把逻辑移到了输入部分D。

并行parallel同步计数器，并行逻辑的优点在于时延较小，对于构建高频、位数多的计数器很有帮助，（可以看到串行的到后面时延越来越大，和之前的行波进位加法器和超前进位加法器可以类比）

我们之前提到计数器是遍历预先规定好的状态序列，也就是说不一定是一个一个往上加。
对于一个任意状态的计数器的设计，就跟设计一个时序电路是一样的。考虑当前状态和下一个状态，列出输入方程设计电路。

第六章 内存Memory

概念：
Memory: A collection of storage cells together with the necessary circuits to transfer information to and from them.
Memory Organization: the basic architectural structure of a memory in terms of how data is accessed.
Random Access Memory(RAM) : a memory organized such that data can be transferred to or from any cell (or collection of cells) in a time that is not dependent upon the particular cell selected.(访问时间和地址无关)
Memory Address: A vector of bits that identifies a particular memory element (or collection of elements).
Memory Operations: operations on memory data supported by the memory unit. 通常包括read和write。
来看memory的一般抽象，注意这里的word指的是memory的组织单元，不是2 bytes，

RAM的种类

  • Static：information stored in latches直到断电之前，信息会被一直存储。
    SRAM，读写速度快，生产成本高，多用于容量较小的高速缓冲存储器。
  • Dynamic：information stored as electrical chargeson capacitors（电容）电容的能量会“泄露”，需要周期性的对存储的内容读写来刷新。
    DRAM，读写速度较慢，集成度高，生产成本低，多用于容量较大的主存储器。
    另一个分类
  • Volatile–如果断电了存储的信息就丢失了。比如上面提到的SRAM和DRAM。
  • Non-volatile–断电了存储的信息就仍然保持。比如ROM
