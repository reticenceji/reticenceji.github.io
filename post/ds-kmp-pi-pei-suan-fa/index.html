DS-KMP

2020-12-07 1 min read # DS

这是一个字符串的寻找子串的匹配算法。
我们首先定义

char S[] = "hellor world"
char T[] = "orl"

暴力匹配

这个算法很好理解。我就随便写一个了。返回第一个找到的子串在主串的index。

for (i=0;i<strlen(S);i++)
{
    for (j=0;j<strlen(T);j++)
    {
        if (S[i+j]!=T[j]) break;    //没找到
    }
    if (j==strlen(T)) return i;     //找到了
}

KMP算法

注意到，如果我们匹配到or，然后发现不匹配时，我们要回溯到r，也就是说我们对r重复比较了。KMP算法就是要消除这个重复比较。
