Linux使用SSH远程登录

2020-11-15 Views Tools | Linux211字1 min read繁

我的需求是本地VMware上开一个虚拟机，然后在主机上连接。两个都是Linux但是版本不同。

主机（虚拟机）的操作

一般来说ssh是默认安装好的。如果没有，用sudo aptitude install openssh-server。这是最普遍的ssh工具。
然后启动ssh-server：/etc/init.d/ssh start
有时候是默认启动的。我们可以用service ssh status看ssh的状态。
因为连接需要知道IP地址，我们用ifconfig命令查看。
inet addr:172.16.57.133 (可以看到这是内网地址啦)

客户机（我的电脑）的操作

当然前提是主机也要安装好ssh。
ssh root@172.16.57.133就可以连接了。然后需要输入密码。

事实上我们可以用过秘钥公钥来简化登录过程。虽然但是，没必要。
