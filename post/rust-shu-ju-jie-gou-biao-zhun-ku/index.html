Option

enum Option<T> {
    Some(T),
    None,
}

Result

enum Result<T, E> {
    Ok(T),
    Err(E),
}

常常和match语句一起使用，match匹配要求枚举是穷尽的。_可以当做default。

match map_b.get(key){
    Some(num) => a = a + f64::from(value * num),
    None => a=a + 0.0
}

Vector

Vec<T>，也叫Vector。允许我们在一个单独的数据结构中存储多于一个的、相同类型的、在内存中彼此相邻排列的值。换句话说，是一个大小可变的数组。可以实现数组、队列、栈等线性表数据结构。

//创建一个数组
let v: Vec<i32> = Vec::new();
//创建一个带有初值的数组
let v = vec![1,2,3];
let mut tit: Vec<String> = vec![String::new();100];
//向数组中添加新的元素，但是这样意味着v是可变的。
//类似的，可以用pop返回并移除最后一个元素
let mut v = Vec::new();
v.push(4);
//获取数组中元素的值，第一种方法如果下标越界会产生panic
let third:i32 = &v[2];
match v.get(2) {
    Some(third) => ...,
    None => ...,
}
//遍历，为了修改可变引用指向的值，必须先使用解引用运算符获取i中的值
for i in &mut v{
    *i += 50;
}

~/.rustup/toolchains/stable-x86_64-unknown-linux-gnu/share/doc/rust/html/std/vec/struct.Vec.html

查看Push的源代码：pub fn push(&mut self, value: T) {...}，传递了自己的一个可变引用。而Rust不允许同时出现一个可变引用和其他任何引用，所以要小心了。

String

Rust 的核心语言中只有一种字符串类型：str，也就是字符串slice，它通常以被借用的形式出现&str。称作 String 的类型是由标准库提供的，而没有写进核心语言部分，它是可增长的、可变的、有所有权的、UTF-8 编码的字符串类型。Rust 标准库中还包含一系列其他字符串类型，比如 OsString、OsStr、CString 和 CStr。
(也就是说在进行不能使用标准库的系统编程时，你的选择就只有str了)

//创建一个字符串，看源代码的话他们的实现是一模一样的。
let mut s = String::from("lo");
let mut s = "lo".to_string();   
//可以通过 push_str 方法来附加字符串 slice，从而使 String 变长。
let mut s = String::from("foo");
s.push_str("bar");
//拼接字符串
let s1 = String::from("Hello, ");
let s2 = String::from("world!");
let s3 = s1 + &s2; // 注意 s1 被移动了，不能继续使用
let s = format!("{}-{}*{}", s1, s2, s3);    //更好的字符串拼接办法
//遍历字符串的办法，字符串是不能直接索引的。
for c in "नमस्ते".chars() {
    println!("{}", c);
}

Map

HashMap<K, V> 类型储存了一个键类型 K 对应一个值类型 V 的映射。
map 可以用于需要任何类型作为键来寻找数据的情况，而不是像 vector 那样通过索引。

//创建一个hashmap
use std::collections::HashMap;
let mut scores = HashMap::new();
//插入键值对
scores.insert(String::from("Blue"), 10);
scores.insert(String::from("Yellow"), 50);
//遍历
for (key, value) in &scores {
    println!("{}: {}", key, value);
}
//只在没有对应内容的时候插入键值对
scores.entry(String::from("Yellow")).or_insert(50);
scores.entry(String::from("Blue")).or_insert(50);

Box

为了构建链表，我们需要Box类型。Rust需要在编译的时候确定类型的大小。

//如果你直接把Box<List>换成List，Rust将无法确定类型的大小，编译失败。
enum List<T>{
    Cons(T, Box<List<T>>),
    Nil,
}
