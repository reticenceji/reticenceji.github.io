Remote-Containers

2020-12-22 2 min read # Tools

本文介绍VSCode插件：Remote - Containers。他的作用是，可以让你使用Docker容器作为功能齐全的开发环境。

准备工作

首先你当然要安装Docker和VS Code，以及上面提到的插件Remote - Containers。
然后我们假设你已经创建了一个Container和并且把一个本地文件夹映射到Container里了。

  1. 首先你可以docker start container_name运行一个container，当然你也可以在在插件中直接打开，如图。当你打开插件的时候，他会自动列出你所有的container，你选择start container就打开了。
  2. 在插件中选择attach container，或者点击左下角的><，选择attach container，如图
  3. 现在开始你的环境就是docker中配置好的环境了！开始你的工作吧。你可能需要打开一个工作区，Open Folder然后选择就OK了，不过是Docker里的文件夹，如果你想把本地的文件夹和Docker关联的话，你可能需要新建一个container，下次需要的时候我再探索探索。
  4. 结束工作的时候，还是类似的点击左下角的><退出连接。然后在插件中stop container。你也可以用命令行。
