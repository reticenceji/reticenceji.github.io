HTML头部元素
标签 内容
<title> 定义文档的标题。
<link> 定义文档与外部资源之间的关系。
<style> 用于为 HTML 文档定义样式信息。
<meta> 元数据（metadata）是关于数据的信息。标签提供关于 HTML 文档的元数据。元数据不会显示在页面上，但是对于机器是可读的。典型的情况是，meta 元素被用于规定页面的描述、关键词、文档的作者、最后修改时间以及其他元数据。标签始终位于 head 元素中。元数据可用于浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他 web 服务。
<script> 标签用于定义客户端脚本，比如 JavaScript。
关于字符

URL 只能使用 ASCII 字符集来通过因特网进行发送。
由于 URL 常常会包含 ASCII 集合之外的字符，URL 必须转换为有效的 ASCII 格式。
URL 编码使用 "%" 其后跟随两位的十六进制数来替换非 ASCII 字符。
URL 不能包含空格。URL 编码通常使用 + 来替换空格。

#####CSS基础语法
CSS 规则由两个主要的部分构成：选择器，以及一条或多条声明。
selector {declaration1; declaration2; ... declarationN }

div#sp_sidebar div#sps_1
{
    width: 160px;
    height: 600px;
    padding: 0;
    margin: 0;
}

根据 CSS，子元素从父元素继承属性。
#####CSS选择器

/*你会发现这里的属性值都没有引号*/
h2,p {color:black;}
/*这是最基本的元素选择器*/
* {color:black;}
/* *号是通配符 */
h2 strong {
     color: blue;
     }
/*这是派生选择器,<h2><strong>被选择的元素</strong></h2>*/
#red {color:red;}
/*这是ID选择器,选择了id是red的元素.也可以和派生合用*/
#sidebar p {
	font-style: italic;
	text-align: right;
	margin-top: 0.5em;
	}
/*id选择器和派生合用*/
.center 
{
    text-align: center
}
/*这是class选择器,选择了class是center的元素.也可以和派生合用*/
[title]
{
    color:red;
    border:5px solid blue;
}
/*这是attribute选择器,选择了有title属性的元素*/
[type="button"]
{
    width:120px;
    margin-left:35px;
    display:block;
    font-family: Verdana, Arial;
}
/*属性选择器,指定了属性的值为button才会被选择*/

JavaScript

JavaScript可以用来做什么?
  1. JavaScript 为 HTML 设计师提供了一种编程工具
    HTML 创作者往往都不是程序员，但是 JavaScript 却是一种只拥有极其简单的语法的脚本语言！几乎每个人都有能力将短小的代码片断放入他们的 HTML 页面当中。
  2. JavaScript 可以将动态的文本放入 HTML 页面
    类似于这样的一段 JavaScript 声明可以将一段可变的文本放入 HTML 页面：document.write("<h1>" + name + "</h1>")
  3. JavaScript 可以对事件作出响应
    可以将 JavaScript 设置为当某事件发生时才会被执行，例如页面载入完成或者当用户点击某个 HTML 元素时。
  4. JavaScript 可以读写 HTML 元素
  5. JavaScript 可以读取及改变 HTML 元素的内容。
  6. JavaScript 可被用来验证数据
    在数据被提交到服务器之前，JavaScript 可被用来验证这些数据。
  7. JavaScript 可被用来检测访问者的浏览器
    JavaScript 可被用来检测访问者的浏览器，并根据所检测到的浏览器，为这个浏览器载入相应的页面。
  8. JavaScript 可被用来创建 cookies
  9. JavaScript 可被用来存储和取回位于访问者的计算机中的信息。

DOM （Document Object Model）

HTML DOM 是关于如何获取、更改、添加或删除 HTML 元素的标准


通过这个对象模型,我们可以操作HTML.
在 DOM 中，所有 HTML 元素都被定义为对象。
属性是您能够获取或设置的值（就比如改变 HTML 元素的内容）。
方法是您能够完成的动作（比如添加或删除 HTML 元素）。
