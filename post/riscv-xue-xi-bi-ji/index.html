2. Instructions

参考中文手册的附录，这里只是随便列了几个提供一下查找的方向。

  • csr( Control and State Register) series：
    • csrr rd, csr x[rd] = CSRs[csr]，读CSR寄存器
    • csrw csr,rd CSRs[csr]=x[rd]，写CSR寄存器
  • b( Branch ) series：
    • beq rs1, rs2, offset if (rs1 == rs2) pc += sext(offset)
    • beqz rs1, offset if (rs1 == 0) pc += sext(offset)
  • l( Load ) series：
    • la rd, symbol x[rd] = &symbol，加载位置
    • lb rd, offset(rs1) x[rd] = sext(M[x[rs1] + sext(offset)][7:0])，字节加载
  • s( Store ) series：
    • sb rs2, offset(rs1) M[x[rs1] + sext(offset) = x[rs2][7: 0]
  • j ( Jump )：
    • j offset pc += sext(offset) 无条件跳转
    • jal rd, offset x[rd] = pc+4; pc += sext(offset) 保存下一条指令的位置并无条件跳转
    • jr rs1 pc = x[rs1]，寄存器跳转
