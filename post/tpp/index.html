TPP

2020-12-17 2 min read # Tools

安装

tpp是一个很有意思的，可以在命令行播放slide的工具。
安装：sudo apt-get install tpp
然后通过写一定格式的tpp脚本，用tpp解析就可以播放了。
效果图：

demo

--author Reticence Ji
--title Demo of TPP
--date today
--bgcolor white
--color black

--header header可以在最上方显示信息
--footer footer可以在最下方显示信息
使用方向键控制，也可以用空格键。
按j实现直接跳转，按h查看帮助。
注意，所有的命令都是以--开头的，需要在行首写，然后用空格隔开。

--newpage 代码段
--heading Code block of TPP
--beginoutput
	代码段可以用 
	"beginoutput" 和 "endoutput" 命令
	在中间写他们的代码
--endoutput
--beginshelloutput
shelloutput可以逐渐的打出命令
$ ls
abc
$$ non_existent
$% ls -l
--endshelloutput


--newpage 格式
--heading heading是标题
--center center是居中
--right right是右对齐
--boldon 
--ulon 
boldon是加粗命令，ulon是下划线命令
直到出现boldoff和--uloff才会变回普通格式
--boldoff 
--uloff
--horline horline是一条分割线

--newpage HUGE
--huge HUGE 

--newpage 逐条输出
想要逐条的输出
---
我们可以使用 - 命令
---
这样在我们按空格之前，都不会输出
