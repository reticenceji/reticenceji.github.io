Latex基本使用教程

2021-06-25 6 min read # Tools

因为老师让我们用Latex写一个survey，学一下latex怎么用。也可以认为是ACM CCS模板的使用简易教程。

安装

LaTeX有很多发型版，TeX Live就是其中一种。TeX Live 是 TUG (TeX User Group) 维护和发布的 TeX 系统，可说是「官方」的 TeX 系统。TeX Live可以保持在跨操作系统平台、跨用户的一致性。安装包如下。
texlive-latex-base和extra都建议安装一下。
xetex其实没有安装的必要，不过也可以安装一下。也是latex编译器的一种，但是和下面的模板不适配。
texlive-fonts-extra如果不安装的话就不能正常使用下面的模板会报错，缺失某些style。
sudo apt-get install texlive-latex-base texlive-latex-extra texlive-xetex texlive-publishers texlive-fonts-extra

Hello world

创建一个hello.tex，输入以下内容

\documentclass{article}
% 这里是导言区
\begin{document}
Hello, world!
\end{document}

然后点击TEX中的Build，目录下就会生成PDF文件。

使用模板

Git上提供了ACM CCS模板。他提供的Makefile和我们之前安装的工具链是不匹配的，所以可以无视他，删除Makefile，PDF，README，选择css-template.tex然后编译就行了。
编译选项要选PDF->Bib->PDF*2，因为这个模板只支持pdftex。
然后正常情况下，我们就看不见警告了。编译之后我们能看见完整的PDF文档。
