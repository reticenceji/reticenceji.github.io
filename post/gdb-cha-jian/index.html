GDB-插件

2021-01-27 1 min read # Tools

插件的安装

修改~/.gdbinit文件：

# 注释掉你不想用的插件
source /data/home/reticence/AAA/gdb/gdbinit-gef.py
#source /data/home/reticence/AAA/gdb/peda/peda.py

gef

gef是一个优秀的GDB插件，帮助我们使用GDB进行逆向和漏洞挖掘。他支持多种架构。
安装非常简单，一般的配置都是Linux默认带有的。你可以按照官网的安装脚本，也可以复制gef.py的源代码到你本地的一个文件，然后在.gdbinit里加上就好了。

peda

peda，也是一个强力的帮助逆向的GDB插件，不过支持的架构少。安装的话直接git clone https://github.com/longld/peda.git，然后在.gdbinit里加上peda.py就好了。
