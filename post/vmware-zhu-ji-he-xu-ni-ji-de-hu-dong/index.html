VMware主机和虚拟机的互动

2020-11-24 Views Tools362字2 min read繁

不知不觉已经用VMware装了5个虚拟机了。基本上主流的操作系统齐活了。你说Mac？我装那玩意儿干啥。
Ubuntu_Server20.04, Ubuntu_Server14.04, Windows7, Windows XP, Dos。

Windows7被我配置成了64位Windows环境，逆向工具+Office办公环境。
WindowsXP是白老师配置的32位Windows环境。逆向工具齐全。
DOS也是白老师配置的16位Windows环境，有SoftICE调试神器。
Ubuntu_Server20.04还是默认配置，Linux环境。
Ubuntu_Server14.04小小配置了一点，Linux环境。
Deepin15.11是我的主机，一个Linux逆向，调试，网络，办公，编程的集大成者。
Docker_oslab是比虚拟机更轻量的Docker环境，GNU-RISCV工具链以及QEMU。

虚拟机装好了，就可以像一个新系统一样用了。下面总结我常用的互动方法。

文件夹共享

VM-Setting-Option-ShareFolder。
把文件夹作为一个设备。

Virtual Disk

vmdk就是我们虚拟机的硬盘。
File-Mount Virtual Disk可以把虚拟机的硬盘挂载在主机的某个目录下。这样我们就可以在主机里查看了。
但是需要注意的是，我们要么把硬盘挂载在主机上，要么挂载到虚拟机上，不能同时。

Server

把我的虚拟机作为一个服务器。具体方法在我的Linux使用SSH远程登录讲述。

Docker的文件夹共享

-v选项可以选择文件夹映射。注意是对Image进行的操作。
