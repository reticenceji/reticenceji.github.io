Rust工具链

2021-05-06 2 min read # Rust

安装 Rust 的主要方式是通过 Rustup 这一工具，它既是一个 Rust 安装器又是一个版本管理工具。
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
Rust采用多版本发布的方式，所以一般会同时有nightly和stable等多个版本。我们一般都是用的nightly版本，因为他的特性更丰富。Rust的工具链由一个3-tuple定义

 Standard release channel toolchain names have the following form:
        <channel>[-<date>][-<host>]
        <channel>       = stable|beta|nightly|<major.minor>|<major.minor.patch>
        <date>          = YYYY-MM-DD
        <host>          = <target-triple>

我们可以用rustup component list --toolchain nightly来查看我们安装的组件，一般来说我们需要下面的内容：

  • Rustc是编译器，一般你已经装好了
  • Cargo是包管理工具。用它来方便的构建工程。
  • RLS是官方提供的标准的编译器增强工具，有了它我们就能在VS code里愉快的coding了。
  • Rustfmt是官方提供的代码格式化工具，有了它我们就能在VS code一键格式化代码了。
  • rust-std-xxx貌似和具体的机器架构有关系。
  • rust-docs
  • rust-analysis
  • clippy 是lint工具，静态分析。

rust-gdb 貌似就是gdb，rust-lldb也就是lldb，加了一层不知道是什么的封装。
