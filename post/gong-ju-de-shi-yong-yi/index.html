工具的使用（一）

2020-07-29 4 min read # Tools

好的工具是安全学习路上的利器。这里仅仅罗列了部分工具。具体的使用可以参考网上的教程、帮助文档、官方Wiki，BIlibili。

网盘链接: https://pan.baidu.com/s/1raHq0mnYLtfAbf64EvDjzQ 提取码: itcq

代码相关工具

工具名 打开方式 功能简述 安装方式
Virtualenv virtualenv venv [--python=某版本] 在一个文件夹下建立虚拟环境;
source venv/bin/activate 打开虚拟环境;
deactivate 关闭虚拟环境		 Python虚拟环境构造 pip下载
VScode 桌面有快捷方式,或者用code命令打开 代码编写,记事本 官网下载
菜鸟工具 (https://c.runoob.com/) 在线运行很多代码，进制转换
Python 命令行打开,python是python2.7.13, python3是python3.5.3, python3.8.5安装在/usr/local/python3.8 脚本编写 官网下载

Web相关工具

工具名 打开方式 功能简述 安装方式
namp 命令行打开 扫描端口 apt
SQLmap ~/AAA/sqlmap-dev,用python3打开 SQL注入 官网源代码
BurpSuite /home/reticence/BurpSuiteCommunity/,桌面有快捷方式 代理工具，重放，爆破 官网下载
postgres数据库 sudo -i -u postgres 然后再输入 psql进入 学习数据库和SQL apt
DirBuster ~/AAA/DirBuser-0.12,用java -jar打开,记得全屏 目录爆破 百度网盘
Githack ~/AAA/GitHack-master/GitHack.py,用python2打开，后面跟.git文件夹的URL git泄露探测，获取.git文件夹的内容 百度网盘、Github
Cknife 安装在 ~/AAA/Cknife,java -jar Cknife.jar 中国菜刀类似,右击创建 百度网盘
translate 命令行输入trans 翻译 apt
Wappalyzer Google-Chrome插件 嗅探服务器，脚本，数据库的类型版本
Subdomain3 Python3运行 子域名扫描器 https://github.com/yanxiu0614/subdomain3/blob/master/README_ZH.md
parameth 在venv虚拟环境中，Python2运行 扫描POST和GET的参数 https://github.com/maK-/parameth

后面这些爆破工具BurpSuite都可以实现啊，可惜是昂贵的professional版。
https://github.com/We5ter/Scanners-Box/blob/master/README_CN.md

逆向相关工具

工具名 打开方式 功能简述 安装方式
IDApro 小白的XP虚拟机 32位和64位静态反汇编工具 暂无
ollydbg 小白的虚拟机 32位动态调试工具 吾爱破解
x64dbg Windows7虚拟机 64位动态调试 吾爱破解
010editor Windows7虚拟机 查看和修改机器码 百度网盘
file file命令 查看文件类型
objdump objdump 目标文件 简单的静态反汇编
gdb-peda 直接用gdb命令就行，安装在家目录~/peda GDB插件 Github
apktool https://github.com/iBotPeaches/Apktool
dex2jar sh d2j-dex2jar.sh -f ~/path/to/apk_to_decompile.apk 作用：将apk反编译成java源码（classes.dex转化成jar文件） https://github.com/pxb1988/dex2jar
jd-gui java -jar jd-gui.jar 查看APK中classes.dex转化成出的jar文件，即源码文件 https://github.com/java-decompiler/jd-gui/releases
