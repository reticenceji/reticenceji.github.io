# 区块链

区块链可以当作是一个分布式数据库。

- 不同节点之间的状态同步通过**网络交互**和**共识协议**完成。
- 对数据库的增删改查的操作就是发送交易。
    - 如果是对智能合约发送的交易，那么交易还要在**区块链虚拟机**中执行。
- **密码学** 对保证数据完整性，身份验证（交易签名）等起作用。
